尚未設定 Google Maps API Key

請把本頁底部 script 內的 YOUR_GOOGLE_MAPS_API_KEY 換成自己的 key，並限制 HTTP referrer 與 API 範圍。

請從 Google 候選地址選擇一筆；若直接輸入完整地址，系統會嘗試自動定位。

此頁的 Google key 會出現在前端，正式使用時請限制網域與 API。正式商業報價仍建議由後端呼叫 Routes API 後回傳 quote_id。
